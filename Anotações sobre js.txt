no js mais atual você pode apenas usar Number e deixalo decidir sozinho

EX:

var numero1 = Number(window.prompt("Digite um numero: "))
var numero2 = Number(window.prompt("Digite outro numero: "))
window.alert("A soma de " + numero1 + " mais " + numero2 + " gerou " + (numero1+numero2))

isso fara a soma de dois numeros corretamente!

aritméticos
5+2  = 7    -soma
5-2  = 3    -subtração
5*2  = 10   -multiplicação
5/2  = 2.5  -divisão real
5%2  = 1    -resto da divisão
5**2 = 25   -potencia(ao quadrado)

lembrar sobre ordem de precedência!!
EX:
5 + 3 / 2 = 6.5
correto:
(5 + 3)/2 = 4

precedência:
()
**
* / %
+ -

simplificação (só funciona se for para a msm variavel)
n = n +  4   n +=  4
n = n -  4   n -=  4
n = n *  4   n *=  4
n = n /  4   n /=  4
n = n ** 4   n **= 4
n = n %  4   n %=  4

operadores de incremento servem para adicionar 1 ou subtrair 1

num = num + 1   num++
num = num - 1   num-- 

pode se usar tbm o simbolo antes de num para assim ele somar ou
subtrair e depois mostrar o valor
